# snakes

仿照 贪吃蛇大作战 的练手项目，留有登录游戏，对战接口，目前实现AI，碰撞检测机制未完成
## introduce

A snake game played on mobile phone or pc, use the "Space" key to speed up the snake which you controlled.

Develop base on layabox by TypeScript

---
